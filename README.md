# 公共物品博弈实验系统 (Public Goods Game)

一个用于行为经济学实验的公共物品博弈游戏系统,包含管理员控制台和玩家界面。

## 🎮 项目简介

公共物品博弈是行为经济学中的经典实验,用于研究合作、利他行为和社会困境。本系统提供了完整的实验管理界面。

### 游戏规则
- 两名玩家各拥有初始代币(默认100)
- 进行多轮博弈(默认3轮,系数分别为1.3、2.0、3.0)
- 每轮玩家决定投入多少代币到公共资源池
- 公共池总金额乘以系数后平均分配给所有玩家
- 游戏结束后,代币最多的玩家获胜

## 📁 项目结构

```
public-good-game/

├── public_goods_game.py     # Python游戏逻辑引擎

```

## 🚀 快速开始

### 方式一:使用Python版本
```bash
python public_goods_game.py
```

### 方式二:使用Web界面
1. 用浏览器打开 `publicgoodgame.html` 进入管理员控制台
2. 分别打开 `player1.html` 和 `player2.html` 进行玩家操作

## 💡 游戏策略

系统内置多种策略示例:
- **贪婪策略**: 投入0代币,期望搭便车
- **合作策略**: 投入所有代币,最大化公共利益
- **平衡策略**: 投入一半代币
- **自适应策略**: 根据系数动态调整投入比例

## 🎯 实验意义

该实验用于研究:
- 合作行为与社会困境
- 利他主义与理性选择的冲突
- 激励机制对合作的影响
- 重复博弈中的策略演化
